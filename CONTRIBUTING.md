# Contributing to GC Stats Riot Relay

First of all, thank you for considering contributing to the Game Changers Stats project! 

To maintain the quality and integrity of the project, please follow these guidelines.

## 🤖 AI Usage Policy

As explained in the organization's core principles, we don't accept raw AI work. AI is a tool, not a replacement for a developer.

- **AI Assistance is OK:** Using tools like Copilot, ChatGPT, or Claude to help you code is perfectly fine.
- **100% AI-Generated PRs are FORBIDDEN:** Submitting code entirely generated by AI without human oversight or understanding is strictly prohibited.

## 💡 How to Contribute

### 1. Start with a Discussion
Before you spend time writing code, **please open a "Feature Request" or "Bug Report" issue.**
- We do not guarantee that every contribution will be accepted.
- Opening an issue allows us to discuss the relevance of the feature and the implementation details to avoid wasted effort.

### 2. Development Workflow
1. **Fork** the repository and create your branch from `main`.
2. Name your branch clearly: `feature/short-description` or `fix/short-description`.
3. Follow the installation steps in the `README.md`.

### 3. Submitting a Pull Request

   - Provide a clear description of what the PR changes or adds.
   - Reference the original Issue (e.g., Closes #123).
   - Before pushing, make sure the checks that CI runs pass locally:
     ```bash
     cargo fmt --check
     cargo clippy -- -D warnings
     cargo test
     ```
